# sycren-design-system

> Shared UI components, design tokens, and Storybook for all Sycren applications.

## Description

Centralized design system containing reusable UI components, design tokens (colors, typography, spacing), and an interactive Storybook instance. Used by `sycren-landing`, `sycren-app`, `sycren-studio`, `sycren-cloud`, `sycren-domains`, and `sycren-security`.

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Storybook
- Lucide React (icons)
